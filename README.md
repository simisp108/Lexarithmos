# Lexarithmos

A bilingual website for Lexarithmos Private Institute in English and Greek.

## Structure:
├─index.html - language selection gateway
├─lexarithmos.html - English Version 
├─lexarithmos-el.html - Greek Version
├─manifest.json
├─robots.txt
├─sitemap.xml
├─css/
│  ├─ main.css - for lexarithmos.html, lexarithmos-el.html
│  ├─ lang-gateway.css - for index.html
├─js/
│  ├─ main.js - for lexarithmos.html, lexarithmos-el.html
│  ├─ lang-gateway.js - for index.html
├─ images/
│  ├─ flag-uk.svg
│  ├─ flag-gr.svg
│  ├─ icon-192.png
│  └─ icon-512.png  
└─ fonts/
   └─ roboto/

## Development
Open in VSC, edit HTML/JS/CSS → preview with Live Server
**Inspect the gateway without redirects:** open `/index.html#preview`.


### Deployment
Runs on https://www.Lexarithmos.ac.cy domain.
